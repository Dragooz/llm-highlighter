## User Roles

Three roles available when inviting a team member:

| Permission | Employee | Admin | Bookkeeper |
|---|---|---|---|
| Have a VISA card | ✅ | If assigned | If assigned |
| View own transactions | ✅ | ✅ | ✅ |
| View all transactions | ❌ | ✅ | ✅ |
| Manage funds & top-ups | ❌ | ✅ | ❌ |
| Manage employees | ❌ | ✅ | ❌ |
| Web dashboard access | ❌ | ✅ | ✅ |
| Approve bills (Payouts) | ❌ | ✅ | ❌ |
| Mobile app access | ✅ | If assigned card | If assigned card |
| Add notes & receipts | ✅ | ✅ | ✅ |
| View all notes & receipts | ❌ | ✅ | ✅ |
| Download statements | ❌ | ✅ | ✅ |

> Choose the right role before inviting. To switch roles after invitation, contact Swipey via Live Chat or WhatsApp.

## Invite Your Team

**Steps:**
1. Admin dashboard → Profile (bottom left) → **Employees** tab → **Invite Employee**
2. Fill in: name, email, phone number
3. Select role (Employee / Admin / Bookkeeper)
4. Optionally assign a card (can be done later)
5. Click **Submit**

Employee receives email invite with setup instructions.

**Employee setup after invite:**
1. Download Swipey app (iOS / Android)
2. Enter email + mobile (do NOT tap "Register new account")
3. Complete eKYC (photo ID + selfie; name must exactly match MyKad/Passport)

> Each invite requires a unique email and phone not previously used on Swipey.

**Common questions:**
- **Employee can't find invite?** Check spam/junk folder
- **Employee shows as "Pending"?** They haven't completed eKYC yet — share the mobile app setup guide
- **Remove employee?** Employee Settings → select → set status to **Inactive** → access immediately revoked

## Get Started with Swipey Mobile App

For employees completing initial setup after receiving invite:

1. Download app: [iOS](https://apps.apple.com/sa/app/swipey-fin-os-for-modern-biz/id1589412706) | [Android](https://play.google.com/store/apps/details?id=app.swipey.co)
2. Enter email address and mobile number
3. Do **not** click "Register new account"
4. Complete eKYC process
5. Once approved, log in and access assigned virtual card

## Update User Role, Department and Status

Admin dashboard → Employee Settings → select employee → Edit → change role / department / status → Save.

Changes apply immediately.
