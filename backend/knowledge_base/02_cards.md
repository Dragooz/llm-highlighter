# Cards

> Everything you need to issue, control and manage your team's cards.

Source: https://help.swipey.co/en/collection/17-cards

## Where Can I Use a Swipey Card?

Swipey cards are VISA prepaid cards accepted at 130M+ merchants worldwide.

**Not accepted by** (prepaid card restriction, not Swipey-specific):
- Google Ads (use one-time payment setup instead)
- Xero, Microsoft Azure, Etsy, ClickUp, DigitalOcean, OpenAI
- Apple Pay, Samsung Pay

## Virtual Cards vs Physical Cards

| | Virtual Card | Physical Card |
|---|---|---|
| Colour | Light blue | Dark blue |
| Plans | All (incl. Freemium) | Growth & Plus only |
| Cards per user (Plus) | Up to 5 | Up to 2 |
| Cards per user (Growth) | Up to 3 | Up to 1 |
| Cards per user (Freemium) | 1 | None |
| Online payments | ✅ | ✅ |
| In-store / PayWave | ❌ | ✅ |
| Instant availability | ✅ | ❌ (post delivery) |
| Access via | Mobile app | Physical card + app |

**To view virtual card details:** App → Cards tab → Show Card Details → verify identity → card number/expiry/CVV shown.

**Physical card first use:** Always use Chip and PIN first to activate PayWave.

## Reset Physical Card PIN

1. Open Swipey mobile app
2. Cards tab → **Reset Card PIN**
3. Authenticate with 6-digit PIN / Face ID / fingerprint
4. Enter OTP
5. Enter and confirm new PIN

---

## Freeze & Reactivate Cards

Freezing blocks all new transactions instantly. Balance unaffected.

**Via mobile app (employee or admin):**
Cards tab → select card → **Freeze Card** / **Reactivate Card**

**Via admin dashboard (admin only):**
Cards → select card → **Freeze Card** / **Reactivate Card**

> Frozen card must be reactivated before it can be terminated.
> Freezing is temporary; terminating is permanent.

---

## Multi-Card

Employees can hold multiple cards for different purposes (e.g., one for ads, one for travel). Number of cards per user depends on plan. Only admins can issue cards.

---

## Spend Customization (Plus Plan Only)

Admin can enable/disable spend per card across 7 categories:
1. Gambling & Massage Parlors
2. Petrol & Car Wash
3. E-Wallet Top Ups
4. ATM Withdrawal
5. Marketing & Advertising
6. Transport & Delivery
7. All Other Spend Categories

**Scenarios:**
- **Restrict specific categories**: Toggle off unwanted categories; all others allowed
- **Allow specific categories only**: Toggle off "All Other Spend Categories" + customize the 6 listed
- **Reset restrictions**: Toggle off Merchant Control entirely

> Changes may take up to 5 working days to apply.

---

## Terminate Cards

Permanent — cannot be undone. Go to: Cards → select card → Terminate.

> Freeze card first to stop spending; terminate only when card is confirmed lost/stolen or employee offboarded.

## Assign Cards to Your Team

See [Getting Started → Assign Cards](./01_getting-started.md#5-assign-cards-to-your-team).

**Common questions:**
- **How many cards can I issue?** Free: 1 virtual; Paid: unlimited virtual + physical per plan
- **Multiple cards per employee?** Yes — separate cards per use case recommended
- **Change spending limit?** Clawback excess funds to master account; adjust as needed
- **Employee leaves?** Employee Settings → set inactive → all cards auto-frozen/deactivated
- **Can employees create cards?** No — admin only
